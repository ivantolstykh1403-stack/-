3 работа  (аппроксимация функций и классификация на нейронных сетях)
3.1. Аппроксимация функций
Рекомендации по реализации (Python, PyTorch):
Генерация данных: для каждой функции (sin, cos, |sin|, |cos|) создайте массив значений X в указанном диапазоне и вычислите Y.
Архитектура сети:
Входной слой: 1 нейрон (значение X).
Скрытые слои: количество и число нейронов задаются в таблице (например, 3 слоя по 32/64/32 нейрона).
Выходной слой: 1 нейрон (предсказанное значение Y).
Функции активации: для скрытых слоёв обычно используют ReLU, для выходного — Linear (для регрессии).
Обучение:
Loss: Mean Squared Error (MSE).
Оптимизатор: Adam или SGD.
Гиперпараметры: количество эпох, learning rate.
